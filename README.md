# toml2json

Convert TOML files to JSON.

# Installation

```
pipx install toml2json
```

# CLI Usage

```
Usage: toml2json [OPTIONS] [TOML_FILES]...

  Converts TOML_FILE(S) to JSON.

Options:
  --help  Show this message and exit.
```
